# Listy

Минималистичный и функциональный планировщик задач, написанный на Python с использованием библиотеки Tkinter. Проект создан для тех, кто ценит простоту, локальное хранение данных и удобный интерфейс.

## Основные возможности

* **Удобное управление:** Добавление, редактирование и удаление задач. Поддержка заголовков и описаний для детального планирования.
* **Умный календарь:** Интерактивный интерфейс с цветовой индикацией (сегодняшний день, выбранная дата и дни с активными задачами).
* **Персонализация:** Мгновенная смена тем (светлая/темная) и локализации (RU/EN) через удобный сайдбар.
* **Конфиденциальность:** Все данные хранятся локально в формате .json. Вы полностью контролируете свою информацию.
* **Скорость:** Широкая поддержка горячих клавиш для управления без мыши.

## Технологии (Tech Stack)

* **Язык:** Python 3.12.3
* **Библиотека GUI:** Tkinter
* **Хранение данных:** JSON
* **Работа с датами:** python-dateutil, babel

## Горячие клавиши (Hotkeys)

| Клавиша       | Действие                                          |
| ------------- | ------------------------------------------------- |
| Ctrl + N      | Создать новую задачу                              |
| Ctrl + S      | Сохранить файл / Сохранить изменения в задаче     |
| Ctrl + T      | Открыть календарь                                 |
| Ctrl + P      | Открыть / закрыть меню                            |
| Delete        | Удалить текущий список задач / Удалить задачу     |
| Стрелки ← / → | Переключение между датами                         |
| Ctrl + C      | Копировать текст задачи (в режиме редактирования) |
| Ctrl + E      | Стереть весь текст в редакторе                    |
| Escape        | Закрыть текущее окно / Выход                      |

## Установка и запуск

### Готовые сборки

Вы можете скачать скомпилированную версию приложения для своей ОС в разделе Releases:

* 🪟 Windows (exe)
* 🐧 Linux (tar.gz)

### Запуск из исходного кода

Для запуска вам понадобится установленный Python 3.12+.

1. Клонируйте репозиторий:

```bash
git clone https://github.com/breeze-git/Listy.git
```

2. Перейдите в папку проекта:

```bash
cd Listy
```

3. Создайте виртуальное окружение:

```bash
python -m venv .venv
```

4. Активируйте его:

```bash
# Linux / macOS
source .venv/bin/activate

# Windows (Command Prompt)
.venv\Scripts\activate

# Windows (PowerShell)
.\.venv\Scripts\Activate.ps1
```

5. Установите зависимости:

```bash
pip install -r requirements.txt
```

6. Перейдите в папку app:

```bash
cd app
```

7. Запустите приложение:

```bash
python main.py
```
